# Installation

## npm
npm i @cruex-de/nuxt-animate.css

## yarn
yarn add @cruex-de/nuxt-animate.css

# Hooking it up

In the nuxt.config.js file add the module in the modules section

~~~~
export default {
  ...
  /*
  ** Nuxt.js modules
  */
  modules: [
    '@nuxtjs/axios',
    '@cruex-de/nuxt-animate.css'
  ]
  ...
}
~~~~
